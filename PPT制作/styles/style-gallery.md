## 详细风格定义

### 01. Apple Bento Grid — 科技发布会型

**参考品牌**: Apple WWDC / OnePlus / Samsung Galaxy

**视觉特征**:
- 纯黑/深灰背景，Bento 宫格分割布局
- 每格容纳单一信息点，留白极多
- 鲜亮功能色（蓝/绿/橙）点缀

**配色方案**:
| 角色 | 色值 |
|------|------|
| 背景主色 | `#000000` |
| 卡片背景 | `#1C1C1E` |
| 主要文字 | `#FFFFFF` |
| 强调蓝 | `#0A84FF` |
| 强调绿 | `#30D158` |
| 强调橙 | `#FF9F0A` |

**字体**: SF Pro Display / 苹方 (标题) + SF Pro Text (正文)
**适合**: 科技产品发布、App 功能展示、硬件发布会

---

### 02. McKinsey 咨询蓝 — 企业权威型

**参考品牌**: McKinsey / BCG / Bain

**视觉特征**:
- 白底蓝字 + 数据图表，信息密度高
- 每页一个核心结论（SCQ 结构）
- 顶部标题行 + 内容区 + 底部注释区

**配色方案**:
| 角色 | 色值 |
|------|------|
| 主背景 | `#FFFFFF` |
| 封面深蓝 | `#002244` |
| 主标蓝 | `#005EB8` |
| 浅蓝强调 | `#4FC1E9` |
| 图表灰 | `#5B616B` |
| 正文黑 | `#1A1A1A` |

**字体**: Georgia (标题) + Arial (正文)
**适合**: 咨询汇报、战略规划、投资人正式汇报

### 03. TED Talk 极简 — 演讲冲击型

**参考品牌**: TED / TEDx

**视觉特征**:
- 每页最多 10 个词，全屏大图或单一数据
- 黑白 + 1 个品牌强调色，居中或左下角单行文字
- 大量留白/负空间

**配色方案**:
| 角色 | 色值 |
|------|------|
| 背景黑 | `#000000` |
| 主文字白 | `#FFFFFF` |
| TED 红 | `#E62B1E` |
| 辅助灰 | `#B0B0B0` |
| 图片遮罩 | `#000000` 50% |

**字体**: Helvetica / Futura, 72pt+ 超大单行标题
**适合**: 公开演讲、行业大会主题演讲、品牌故事

### 15. 发布会全图大字 — 舞台演讲型

**参考品牌**: 罗振宇「时间的朋友」跨年演讲 / Apple Keynote / 小米发布会 / TED 全图变体

**视觉特征**:
- 每页全屏底图（实景摄影/AI生图/模糊处理），文字叠加在图上
- 每页仅 1-3 行大字（≤30 字），演讲者口述补充全部细节
- PPT 是「视觉锚点」而非「信息载体」，观众看屏幕只需 2 秒
- 4 种子页面类型轮替使用，保持节奏感

**4 种子页面类型**:

| 子类型 | 底图策略 | 文字策略 | 示例 |
|--------|---------|---------|------|
| **数据冲击** | 高斯模糊实景照 | 左侧标题 + 右侧超大数字阵列 | "73% 交易额上涨" |
| **品牌故事** | 高清叙事照片（场景即故事） | 品牌 Logo + 1 句 slogan | MINISO 巴黎街头 |
| **章节转场** | AI 生成意境图（情绪隐喻） | 章节号 + 1 行问题/标题 | 海底鲸鱼 = 探索未知 |
| **人物引言** | 人物肖像 + 模糊场景 | 引号包裹金句 + 署名 | "混凝土行业本质是个生鲜行业" |

**配色方案**:
| 角色 | 色值 |
|------|------|
| 文字主色（深底图） | `#FFFFFF` |
| 文字主色（浅底图） | `#1A1A1A` |
| 强调色（数字/关键词） | `#C9A84C` 金色 或品牌色 |
| 图片遮罩（深色） | `rgba(0,0,0,0.35-0.50)` |
| 图片遮罩（浅色） | `rgba(255,255,255,0.30-0.50)` |
| 文字阴影 | `0 2px 8px rgba(0,0,0,0.5)` |

**字体**: Arial / Helvetica, 60-120pt 超大标题，粗体
**信息密度**: 极低（每页 ≤30 字，全靠演讲者口述）

**底图来源混合策略**:
1. AI 生成（Gemini）— 意境抽象图、视觉隐喻、无法拍摄的场景
2. 实景摄影（Pexels/Unsplash）— 城市、自然、人物
3. 品牌素材 — 产品截图、用户提供的照片
4. 模糊处理 — 任何照片加高斯模糊后当「质感底图」

**文字可读性保障**（必须至少用 1 种）:
- 半透明遮罩：图上加 `rgba(0,0,0,0.4)` 黑色蒙版后叠白字
- 底图模糊：高斯模糊让图退为纹理
- 文字阴影：`text-shadow: 0 2px 8px rgba(0,0,0,0.5)`
- 局部留白：底图构图本身留出文字安全区
- 色块衬底：文字下方加半透明色条

**与 TED Talk 极简(03)的区别**:
- 03 有大量纯黑/纯白页面（无底图），本风格每页都有全屏底图
- 03 文字更少（可能只有 1 个词），本风格允许 2-3 行
- 03 图片是偶尔的冲击元素，本风格底图是常态基底
- 本风格制作成本更高（需要 20+ 张底图）

**适合**: 跨年演讲、品牌发布会、企业年会、大型行业峰会主题演讲

## 使用方式

制作 PPT 时告诉我：
1. **选风格**: 报编号（01-15）或风格名
2. **描述感觉**: 说"高端科技感"或"温暖自然"，我来推荐
3. **混搭**: 如"封面用 06 玻璃态，内容用 12 Notion 极简"
4. **自定义**: 指定品牌色，我来适配最接近的风格
